Sistem informasi pelaporan kejadian banjir untuk masyarakat Kabupaten Cirebon ke BPBD.

## ✨ Fitur Utama

- ✅ Form laporan dengan upload foto
- ✅ GPS otomatis (OpenStreetMap/Leaflet)
- ✅ Dashboard admin lengkap
- ✅ Tracking status real-time
- ✅ Grafik statistik dengan Chart.js
- ✅ Responsive design

## 🚀 Cara Install

1. Clone repository
2. Import `flood_report_db.sql` ke MySQL
3. Konfigurasi `config/database.php`
4. Jalankan di browser
